Go - Package

package是golang中组织代码的基本单元，所有代码都应该在package中。main函数应该被定义在main包中，它是程序的入口，其他package不能包含main函数。同一个package下的代码共享空间，可以直接互相调用，包括小写字母开头的成员，通过import关键字引入package。import语句写在go源代码文件的开头。

$ go mod init look/example
go: creating new go.mod: module look/example

$ touch main.go

在工作目录下执行上面的命令，生成一个go.mod文件，即创建一个module。

package main

import "fmt"

func main() {
    fmt.Println("The main function is the entry point of the program.")
}

$ go run main.go

可以公开type,global variables, constants, functions, methods以便在其他包中使用它们。要公开它们，名称要以大写字母开头，如果不想共享它们，则名称要以小写字母开头。

接下来我们创建一个greeting文件夹，在文件夹内创建greeting.go文件，添加如下代码。

package greeting

import "fmt"

func Greeting() {
	hello()
}

func hello() {
	fmt.Println("Hello, World!")
}

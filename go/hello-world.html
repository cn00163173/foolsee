Go by Example: Hello World

第一个示例程序将会在控制台打印经典的Hello, world!，以下是源代码：

package main

import "fmt"

func main() {
    fmt.Println("Hello, world!")
}

我们也可以使用go build命令将程序构建成二进制可执行文件。

$ ./hello-world
